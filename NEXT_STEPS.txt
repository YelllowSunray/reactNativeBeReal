🚀 WHAT YOU NEED TO DO (3 SIMPLE STEPS):

┌─────────────────────────────────────────────────────────────────────┐
│ STEP 1: Get reCAPTCHA v3 Site Key (5 minutes)                      │
├─────────────────────────────────────────────────────────────────────┤
│ 1. Go to: https://console.firebase.google.com/                     │
│ 2. Select project: reactnativevideobereal                          │
│ 3. Click "App Check" in sidebar                                    │
│ 4. Click "Apps" tab                                                │
│ 5. Register web app with "reCAPTCHA v3"                            │
│ 6. Copy the Site Key (starts with 6L...)                           │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ STEP 2: Add Site Key to Code (2 files)                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│ FILE 1: firebase.js (line 6)                                       │
│ Replace:                                                            │
│   const RECAPTCHA_V3_SITE_KEY = '6LdYOUR-SITE-KEY-HERE';           │
│ With:                                                               │
│   const RECAPTCHA_V3_SITE_KEY = '6LcYOUR-ACTUAL-KEY';              │
│                                                                     │
│ FILE 2: web/index.html (line 49)                                   │
│ Replace:                                                            │
│   const RECAPTCHA_V3_SITE_KEY = '6LdYOUR-SITE-KEY-HERE';           │
│ With:                                                               │
│   const RECAPTCHA_V3_SITE_KEY = '6LcYOUR-ACTUAL-KEY';              │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ STEP 3: Test It!                                                   │
├─────────────────────────────────────────────────────────────────────┤
│ Test on web:                                                        │
│   npm run web                                                       │
│                                                                     │
│ Test on mobile (requires development build):                       │
│   npx expo prebuild                                                │
│   npx expo run:ios  (or run:android)                               │
└─────────────────────────────────────────────────────────────────────┘
